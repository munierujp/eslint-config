# @munierujp/eslint-config

ESLint Shareable Config for JavaScript

## Installation

Install `@munierujp/eslint-config`.

```sh
npm i -D @munierujp/eslint-config
```

## Usage

Add `@munierujp` to `extends` of your ESLint config.

```json
{
  "extends": [
    "@munierujp"
  ]
}
```
